# Usage

It reads a [initial state pattern](https://conwaylife.com/wiki/Pattern) of the cells from a `<pattern>.txt` in `patterns/`.
The pattern itself is a [plaintext file](https://conwaylife.com/wiki/Plaintext).

```sh
git clone https://github.com/JDeepD/jacob.git
cd jacob
mix deps.get
mix scenic.run -- <pattern>.txt
```

> Note: Scenic has a few dependencies which needs to be installed before running it: https://hexdocs.pm/scenic/install_dependencies.html
